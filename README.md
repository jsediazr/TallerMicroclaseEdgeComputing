# Taller: Detección de Objetos con YOLOv8

## Pasos

### 1. Instalar la librería
pip install ultralytics

### 2. Estructura de carpetas
taller/
├── detector.py
├── gato_playero.webp
└── resultados.txt  ← se genera al correr el código

### 3. Utilizar el código base
Solo modificar la parte marcada con TU CÓDIGO VA AQUÍ

### 4. Correr el código
python detector.py gato_playero.webp

### 5. Resultado esperado en resultados.txt
Imagen: gato_playero.webp
Etiquetas detectadas: [lista de objetos que YOLO detectó]

## Lo que entregan en el fork
taller/
├── detector.py        ← con su código completo
├── gato_playero.webp  ← la misma imagen que les diste
└── resultados.txt     ← generado al correr el código
